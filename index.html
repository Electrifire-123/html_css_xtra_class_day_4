1.Inline

Inline : It is a Property in display which will occupy how much the content of the elements is required, It will always come in same line

2.Block

Block:Property in display which will occupy complete width and it will come in next line

2.Inline-Block

Inline-Block:Property in display It is a combination of both inline and block, we can apply both height and width and margin and padding

2.None

None:which will completely remove our element from the document or our browser screen

2.Visibilty Hidden

Visibilty Hidden:It will hide the content or elemtnt but the space is going to be there.

2.Flex

Flex:It is a one directional display property either we can hae row or column only and we can manupulate our items.
